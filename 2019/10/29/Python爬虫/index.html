Scrapy中POST请求

Fiddler中抓取到的请求网址在postman中的使用方式：

  1. 首先获取Fiddler中的网址，即右键——>复制——>仅复制URL

  2. Postman中，粘贴刚复制好的URL，将请求方式修改为POST

  3. 然后在Fiddler中获取请求的body字段：TextView中的文本

  4. Postman中在body中的raw字段下粘贴复制好的body字段
